🔧 main.py

from fastapi import FastAPI, Request
from pydantic import BaseModel
from fastapi.responses import JSONResponse

app = FastAPI()

@app.get("/status")
async def get_status():
    return {"status": "running", "source": "replit"}

class GenRequest(BaseModel):
    prompt: str

@app.post("/generate")
async def generate_response(req: GenRequest):
    # Placeholder response logic
    return {"response": f"Generated text for: {req.prompt}"}

🧪 openapi.json (for GPT plugin integration or Swagger UI)

This is optional unless you’re wiring a Custom GPT.

{
  "openapi": "3.0.0",
  "info": {
    "title": "Replit FastAPI Backend",
    "version": "1.0.0"
  },
  "paths": {
    "/status": {
      "get": {
        "summary": "Health Check",
        "responses": {
          "200": {
            "description": "Service Status",
            "content": {
              "application/json": {
                "example": {
                  "status": "running"
                }
              }
            }
          }
        }
      }
    },
    "/generate": {
      "post": {
        "summary": "Generate Text",
        "requestBody": {
          "content": {
            "application/json": {
              "schema": {
                "type": "object",
                "properties": {
                  "prompt": {
                    "type": "string"
                  }
                },
                "required": ["prompt"]
              }
            }
          }
        },
        "responses": {
          "200": {
            "description": "Generated Response",
            "content": {
              "application/json": {
                "example": {
                  "response": "Generated text for: your input"
                }
              }
            }
          }
        }
      }
    }
  }
}

🚀 How to Use on Replit
	1.	Create a new Replit → Python (or Empty Repl)
	2.	Paste all files
	3.	Replit will auto-detect and install dependencies
	4.	Your backend will run on https://<your-repl>.repl.co
	5.	Test GET /status or POST /generate via Postman or browser

⸻

🧩 Optional: Connect to GPT (via Tool / Plugin)
	•	Add the openapi.json URL (hosted via Replit)
	•	Set auth to none
	•	Now your GPT has a live API tool powered by Replit
